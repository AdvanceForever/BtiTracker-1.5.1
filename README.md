Requirements
================
1: Memcached Server
<br />
2: Memcached Extension (Not to be confused with Memcache)
<br />
3: PHP 5.3 > Above
<br />
4: Apache Web Server 2 > Above or NGINX
<br />
5: MySql Server
<br />
6: MySqli Extension
<br />
7: zlib
<br />
8: mbstring
<br />
9: gd2
